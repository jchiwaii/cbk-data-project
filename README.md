# Kenya Economic Dashboard

Interactive dashboard analyzing Kenya's macroeconomic data from the Central Bank of Kenya (CBK).
Full [report](https://medium.com/@chiwai.kiriba/the-shilling-the-strain-and-the-dream-kenyas-economy-in-numbers-2dd273d03d05)
## Features

- **GDP Analysis** - Nominal/Real GDP trends and growth rates
- **Debt Analysis** - Public debt composition (domestic vs external)
- **Inflation Tracking** - Historical trends with CBK target bands
- **Fiscal Overview** - Government revenue vs expenditure
- **Correlations** - Relationships between economic indicators

## Data Sources

- Annual GDP (2000-2024)
- Public Debt (1999-2025)
- Inflation Rates (2005-2025)
- Revenue & Expenditure (1999-2025)
- Domestic Debt by Instrument (1999-2025)

## Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Run the dashboard
python app.py
```

Open http://localhost:8050 in your browser.

## Tech Stack

- **Dash** - Web framework
- **Plotly** - Interactive charts
- **Pandas** - Data processing
